# Parts you will need
* Raspberry Pi 4
* micro sd card
* power bank
* l293d quad half-h bridge motor driver
* rover of some sort
* usb webcam
* duct tape
* HC-SR04 ultrasonic distance sensor
* breadboard
* jumper wires
